== Description ==


This plugin allows you to increase your users and sales thanks to coupons!

<b>WHAT THIS PLUGIN DOES</b>
If users are not logged in, if they visit cart page will see a popup who asks if they want a coupon. If user give his email address he will access to coupon and he will be registered as customer.

<b>HOW TO USE</b>
Install this plugin in your wordpress. In WP menu will appear a new section "NS Coupon Customer" clicking it you will see the option page where you can find some fields:
- Coupon Name: you must insert your WooCommerce coupon text previously created. 
- PopUp Title and Subtitle: insert, if you want, you marketing phrases here. This helps you to increase sales and registrations.
- Enable Privacy Policy: enable this checkbox if you want inform your customer about your privacy policy.
- URL Privacy Policy: if "Enable Privacy Policy" field is checked will appear another field where you can insert your link to your privacy policy page.

== Installation ==

1. Upload the entire 'ns-coupon-to-become-customer' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'NS Coupon Customer'
